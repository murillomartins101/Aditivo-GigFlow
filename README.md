# GigFlow

Interface principal da plataforma GigFlow agora em **React + TypeScript + Vite + Tailwind CSS**.

O design foi baseado no projeto original do Figma:
https://www.figma.com/design/8XDaYbuMlrjdTDBUIh6fXw/GigFlow-app-interface-design

## Stack atual

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Componentes UI modernos (Radix + Lucide + Sonner)

## Como rodar

```bash
npm install
npm run dev
```

A aplicação será iniciada em ambiente de desenvolvimento via Vite.

## App legado (Streamlit)

O app antigo em Python/Streamlit foi preservado em:

- `legacy/Aditivo GigFlow.py`
- `legacy/gigflow/`
- `legacy/.streamlit/`
- `legacy/runtime.txt`
